大于仿函数：

template T greater //大于仿函数

心得思考：

添加头文件不可忘记，要各自对应，在使用sort(三个参数)的时候，第三个参数，通常为自定义的 函数对象（仿函数）。因为默认使用sort(两个参数)的使用，其底层也是默认调用的 系统的默认的内建函数对象less<>()(六个关系运算类函数对象的倒数第二个)。

代码演示：

创建一个控的项目，再添加一个空的main.cpp,在里面添加如下代码:

#pragma once

#include <iostream>
using namespace std;
#include <vector>
#include <algorithm>  //算法
#include <functional>  //内建函数对象

int main()
{
	vector<int> v;
	v.push_back(10);
	v.push_back(30);
	v.push_back(40);
	v.push_back(20);
	v.push_back(60);
	v.push_back(50);

	sort(v.begin(), v.end(), greater<int>());   //内建关系函数对象（无名 临时对象）greater<int>()   大于
	for_each(v.begin(), v.end(), [](int val) {cout << val << endl; });
	
	return 0;
}

运行演示：

默认排序是从小到大，而这里结果是从大到小，复合结果的预期。

细节方面：
