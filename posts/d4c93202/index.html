项目在Linux平台发布：

步骤：Linux发布免安装的运行程序

（1）同样，将完整的ChineseChess项目在Linux用Qt Creator编译运行Release版本。

（2）但是编译不通过，会出现两个错误提示。

error:-1: error: cannot find –lGL和
error:-1: error: collect2: error: ld returned 1 exit status。

原因：是这是由于 Qt5.0 以上的版本默认将OpenGL加入了工程，但是在Linux的机器上没有安装OpenGL，所以只需要在机器系统安装OpenGL即可，此时打开终端，运行

sudo apt-get install libgl1-mesa-dev

即可，再次编译运行。即可成功。

（3）同样将生成的Release文件夹下ChineseChess文件放到一个英文文件夹(eg: qwer)里面.


说明: 即是 创建一个 “qwer” 文件夹，将 build-ChineseChess-unknown-Release 里面的 ChineseChess 文件复制到 “qwer” 下， 然后创建两个空的脚本文件 pack.sh 和 Cpack.sh ;脚本内容如下


（4）在上一步的文件夹中新建文件pack.sh。内容如下：

#!/bin/sh  
exe="ChineseChess" #你需要发布的程序名称
des="/home/xmuli/Desktop/temp/qwer" #创建文件夹的位置
deplist=$(ldd $exe | awk  '{if (match($3,"/")){ printf("%s "),$3 } }')  
cp $deplist $des

（5）在此目录下再新建一个ChineseChess.sh文件（文件名必须与可执行文件名字一样）， 文件内容如下（不需要修改）：

#!/bin/sh  
appname=`basename $0 | sed s,\.sh$,,`  
dirname=`dirname $0`  
tmp="${dirname#?}"  
if [ "${dirname%$tmp}" != "/" ]; then  
dirname=$PWD/$dirname  
fi  
LD_LIBRARY_PATH=$dirname  
export LD_LIBRARY_PATH  
$dirname/$appname "$@"

（6）打开终端，执行命令如下

./pack.sh

会自动ChineseChess.sh 讲所有依赖全部放入这个文件夹里面

（7）将此目录打包发布即可，点击即可以执行。
