启动盘制作工具：

深度启动盘制作工具是深度科技团队开发的一款系统启动盘制作工具，具有界面简洁、操作简单的特点，它可以帮助用户简单便捷地制作deepin启动U盘。

项目地址：

  • 源码地址：https://github.com/linuxdeepin/deepin-boot-maker
  • 国际化地址：https://www.transifex.com/linuxdeepin/deepin-boot-maker/
  • Windows下载地址： http://cdimage.deepin.com/applications/deepin-boot-maker/windows/
  • MacOS下载地址： http://cdimage.deepin.com/applications/deepin-boot-maker/macos/
  • uos / deepin： 系统自带

开启开发者模式：

在 控制中心 里面设置； 然后激活， 重启电脑；


换源：

sudo cp ./sources.list  /etc/apt/sources.list

安装 git:

sudo apt update
sudo apt install git

#------------------------------------------------------------
git config --global user.name "xmuli"  # 本地机器设置全局的姓名和邮箱
git config --global user.email "xmulitech@gmail.com"

ssh-keygen -t rsa -C "xmulitech@gmail.com"  # 生成密钥

cat ~/.ssh/id_rsa.pub  # 显示公钥

# 进入 github / gitlab 的 setting-ssh and gpg keys 去创建新的值

# 成功，可以正常使用

git config --global commit.gpgsign true # 设置 gpg 签名(若有)

安装 QtCreator：

sudo apt install qtcreator     #安装Craetor IDE
sudo apt install qt5-default   #安装qt的配置
sudo apt install libdtkwidget-dev  #安装dtk开发所需要环境(实际可以不需要)
sudo apt install qtcreator-template-dtk  #安装 Dtk Widgets Application工程模板

cd /usr/src
sudo apt source qt5-default    #qt源码包，此会安装在当前所处目录下，建议放在/usr/src 目录再执行
  
#==============================================================
# 也可以将上面的合成一个命令
sudo apt install qt5-default qtcreator cmake

# 若是测试 QtCreator 运行空醒目失败，报时间错误，则重新安装 qmake （默认不需要执行）
sudo apt install qt5-qmake --reinstall

安装 zsh：

  • 官网：ohmyzsh
#下载 zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"  

cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc  #拷贝模板
vim ~/.zshrc  #修改主题为 ZSH_THEME="cloud “

chsh -s /bin/zsh   #从bash切换到zsh
chsh -s /bin/bash  #从zsh切换到bash\


#------------------------------------------------------------
# 可以在 .zshrc 后面添加如下

PATH="$PATH:./node_modules/.bin"
export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles

export GPG_TTY=$(tty)

#设置 tree 显示中文且带有颜色
alias tree='tree -CN'

#设置行号
set number

配置 vim 主题(one)：

  • one 主题官网： vim-one
下载 one.vim 文件 # https://github.com/rakr/vim-one/tree/master/colors 为下载网址

mkdir ~/.vim/colors/  # 没有该文件夹就创建
sudo cp ./one.vim ~/.vim/colors/ # 复制到 vim 主题下

vim ~/.vimrc # 修改 vim 配置文件，没有就创建；在 ~/.vimrc 加入如下三行代码后保存退出。「使用 one 主题的 dark 风格，且按下 delete 按键会删除字符」
colorscheme one
set background=dark
set backspace=2

下载 gitlab 仓库地址：

git remote -v

git clone git@github.com:linuxdeepin/dtkwidget.git
git remote add self xxxxxxx

git clone git@github.com:linuxdeepin/qt5integration.git
git remote add self xxxxxxx

安装常用软件：

# gammaray： 安装 Qt 调试工具 
# flameshot 安装截图工具
# com.notepadqq.notepadqq 安装 notepadqq； "设置-首选项-外观"选择duotone-light
# gitg git 可视化工具


# 商店里面安装讯飞输入法、typora、wps（或官网里面的）
sudo apt install gammaray flameshot com.notepadqq.notepadqq tree gitg

scp -r ./amd64 wang@10.20.32.137:~/ # 使用 scp 传送文件
